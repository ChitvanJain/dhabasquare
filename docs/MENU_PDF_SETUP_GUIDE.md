# Dhaba Square Menu PDF Setup Guide

## For: Restaurant Manager (Non-Technical)

This guide explains how to set up and update your restaurant menu on the website using Google Drive and a PDF file.

---

## ⚡ Quick Overview

- **Where is the menu?** → Google Drive (cloud storage)
- **How do customers access it?** → Website button that opens the PDF
- **How often can you update it?** → As often as needed (takes 2 minutes)
- **Do you need technical skills?** → NO, just basic file management

---

## 🚀 SETUP (One-Time Only)

### Step 1: Create a Google Drive Folder

1. Go to **Google Drive** (https://drive.google.com)
2. Log in with your Google account (create one if needed)
3. Click **"New"** → **"Folder"**
4. Name it: **"Dhaba Square Menu"**
5. Press Enter to create it

---

### Step 2: Upload Your Menu PDF

1. Open the **"Dhaba Square Menu"** folder (double-click it)
2. Click **"New"** → **"File upload"** OR drag your PDF into the folder
3. Select your menu PDF file from your computer
4. Wait for upload to complete (you'll see a checkmark)

---

### Step 3: Get the File ID

1. Right-click on the uploaded PDF
2. Click **"Share"**
3. Click **"Change"** (next to "Restricted")
4. Select **"Viewer can access"** (allows anyone with the link to view)
5. Click **"Share"**
6. Copy the link (it looks like: `https://drive.google.com/file/d/XXXXXXXXXXXXXXXX/view`)

**Extract the ID from the link:**
- The ID is the long string between `/d/` and `/view`
- Example: `https://drive.google.com/file/d/`**`1A2B3C4D5E6F7G8H9I0J`**`/view`
- Your ID = `1A2B3C4D5E6F7G8H9I0J`

---

### Step 4: Give Your ID to the Developer

Send the developer (or Chitvan) the **FILE ID** you copied above.

**They will:**
- Update the website code to use your Google Drive PDF
- Test that the menu displays correctly
- That's it! Website is live with your menu.

---

## 🔄 UPDATE YOUR MENU (Whenever You Need)

### Method 1: Replace the File (RECOMMENDED)

This is the easiest way:

1. Go to **Google Drive**
2. Open **"Dhaba Square Menu"** folder
3. Right-click on the old PDF file
4. Click **"Delete"** or **"Remove"**
5. Click **"New"** → **"File upload"**
6. Select your **new menu PDF**
7. Wait for upload to complete
8. **DONE!** Website automatically shows new menu (might take 5 minutes)

### Method 2: Overwrite the Same File

1. Go to **Google Drive**
2. Right-click on the menu PDF
3. Click **"Open with"** → **"Google Drive"** (or upload a new file with the same name)
4. Replace the content
5. Website updates automatically

---

## ✅ How Website Visitors See the Menu

When customers visit your website:

1. They scroll to the **"Menu"** section
2. They see your **PDF displayed inline** (can scroll through it)
3. They have 2 buttons:
   - **📥 Download Menu PDF** → Saves to their computer
   - **🔗 Open in New Tab** → Full-screen view of PDF

---

## 🔐 Permissions & Access

- ✅ **Anyone with the link can view** the menu
- ❌ **No one can edit** your PDF without permission
- ✅ **Everyone sees the latest version** you uploaded
- ✅ **Mobile-friendly** (works on phones too)

## 📞 Troubleshooting

### "The PDF won't show on the website"
→ Wait 5-10 minutes for Google to process the change

### "The file is taking too long to upload"
→ Check your internet connection or try a smaller PDF file

### "I want to share editing rights with staff"
→ Go to the **"Dhaba Square Menu"** folder, right-click, **Share**, add staff email, set as **"Editor"**

### "I accidentally deleted the PDF"
→ Go to Google Drive **Trash** → Right-click on PDF → **Restore**

## 💡 Tips

**Best Practices:**
- Keep the PDF file name simple: `Dhaba-Square-Menu.pdf`
- Update the PDF before big events/season changes
- Keep original file backed up on your computer
- Use good quality PDF (make menu clearly readable)

**File Size:**
- Recommended: Under 5MB (loads faster)
- Maximum: 25MB (Google Drive limit)
- If PDF is large: Compress it using online tools

**Format Tips:**
- Use high contrast (dark text on light background recommended)
- Include prices clearly
- Add branch-specific items if different between locations
- Consider adding QR codes for social media
