# 3-STEP Menu Update Process
## For: Restaurant Manager at Dhaba Square

---

## 🎯 Goal
Update your restaurant menu on the website in **2 minutes** without needing a developer.

---

# ⚡ 3 SIMPLE STEPS

## STEP 1️⃣: Upload New Menu
**Time: 1 minute**

1. Go to: **https://drive.google.com**
2. Log in (or create Google account if you don't have one)
3. Find your folder: **"Dhaba Square Menu"**
4. Delete the old menu PDF (right-click → Delete)
5. Upload your new menu PDF (drag it into the folder)
6. Wait for upload to finish (green checkmark appears)

**✅ Done!**

---

## STEP 2️⃣: Website Updates Automatically
**Time: 5 minutes (automatic)**

That's it! Your new menu appears on the website within 5-10 minutes.

No other steps needed. Seriously.

**How it works:**
- Google Drive tells the website "new menu is ready"
- Customers see the new menu when they visit
- Download/View buttons work with the new PDF

---

## STEP 3️⃣: (Optional) Share with Staff
**Time: 1 minute**

If you want other staff to help update the menu:

1. Go to: **https://drive.google.com**
2. Right-click **"Dhaba Square Menu"** folder
3. Click **"Share"**
4. Add their email address
5. Set permission to **"Editor"**
6. They can now upload menus too

# 🆘 Stuck? Common Issues

| Problem | Solution |
|---------|----------|
| Can't find folder | Go to: https://drive.google.com (search for "Dhaba Square Menu") |
| Upload is slow | Check internet. Large files take longer. |
| Website still shows old menu | Wait 10 minutes for Google to update. Then refresh website. |
| Don't have Google account | Create free account at https://accounts.google.com |
| Accidentally deleted menu | Go to Google Drive **Trash**, right-click, click **Restore** |

# 💡 TIPS

**Best Practices:**
- Update menu once a week (or when prices change)
- Keep menu PDF file named: `Dhaba-Square-Menu.pdf`
- Always keep a backup on your computer
- Make sure text is clear and readable

**Format Tips:**
- Keep PDF file size under 5MB (faster loading)
- Use large fonts (customers view on phone)
- Include prices clearly
- Add photos if possible (looks professional)

---

# 📞 Need Help?

**If something doesn't work:**
1. Wait 10 minutes (Google sometimes takes time)
2. Refresh your browser with F5
3. Try a different web browser (Chrome, Safari, Firefox)
4. Contact the developer with a screenshot
